# Aplikasi Pengelolaan Keuangan UMKM

**Kelompok 5 | UAS Pemrograman Web**

Aplikasi web untuk membantu pelaku UMKM mencatat transaksi keuangan, mengelola produk & stok, serta menyediakan laporan keuangan (Laba/Rugi, Arus Kas, Piutang).

## Tech Stack
Laravel 13 · MySQL · Blade · Tailwind CSS · Laravel Breeze
